Haben Sie eine Idee 💡! Speichern Sie den aktuellen Anfang der Teilzeichenkette und die längste Länge in Variablen. Erstellen Sie ein Wörterbuch, um den letzten Index jedes Zeichens zu speichern. Iterieren Sie über die angegebene Zeichenkette, aktualisieren Sie gegebenenfalls die Startposition (nächster Index nach dem letzten Vorkommen des aktuellen Zeichens) und berechnen Sie die Länge der einzelnen Zeichen.

Ok, womit soll ich beginnen?

Deklarieren Sie eine Variable für den Anfang der Teilzeichenkette, die längste Länge und ein Wörterbuch für die Zeichen. 

start = res = 0
chars = {}

Iterieren Sie über die Indizes und Zeichen der angegebenen Zeichenkette mit der Funktion enumerate(). 

for ind, char in enumerate(s):
    ...
Was ist als nächstes zu tun?

Zunächst müssen Sie prüfen, ob das aktuelle Zeichen im Wörterbuch vorhanden ist (es wurde bereits zuvor in der angegebenen Zeichenfolge gefunden). 

if char in chars:
    ...

Wenn dies der Fall ist, müssen Sie die Position start so aktualisieren, dass sie das Maximum des aktuellen Wertes und den Index direkt nach dem vorherigen Vorkommen des aktuellen Zeichens ist. 

start = max(start, chars[char] + 1)
Ich brauche noch einen Hinweis.

Jetzt müssen Sie den letzten Index des aktuellen Zeichens im Wörterbuch setzen (aktualisieren). 

chars[char] = ind
Wie kann man die aktuelle Länge der eindeutigen Teilzeichenkette zählen?

Ja, Sie haben den gespeicherten Wert res und Sie haben den aktuellen Index und den Startwert (den Anfang der Teilzeichenkette mit eindeutigen Zeichen). Ermitteln Sie die Differenz zwischen Index und Startwert und aktualisieren Sie res mit dem Maximalwert. 

res = max(res, ind - start + 1)
Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur alle vorherigen Hinweise verbinden, das Endergebnis zurückgeben und Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

def longest_substr(s: str) -> int:

    start = res = 0
    chars = {}
    for ind, char in enumerate(s):
        if char in chars:
            start = max(start, chars[char] + 1)
        chars[char] = ind
        res = max(res, ind - start + 1)

    return res
