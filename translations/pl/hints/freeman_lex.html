Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę przechowywać aktualny początek podciągu i najdłuższą długość w zmiennych. Proszę utworzyć słownik do przechowywania ostatniego indeksu każdego znaku. Iteracja po podanym ciągu, aktualizacja pozycji początkowej w razie potrzeby (następny indeks po poprzednim wystąpieniu bieżącego znaku) i obliczenie długości unikalnych znaków.

Ok, od czego powinienem zacząć?

Proszę zadeklarować zmienne dla początku podciągu, najdłuższej długości i słownik dla znaków. 

start = res = 0
chars = {}

Iterować po indeksach i znakach podanego ciągu za pomocą funkcji enumerate(). 

for ind, char in enumerate(s):
    ...
Co robić dalej?

Przede wszystkim należy sprawdzić, czy bieżący znak jest obecny w słowniku (czy został już wcześniej napotkany w danym ciągu znaków). 

if char in chars:
    ...

Jeśli tak, należy zaktualizować pozycję start tak, aby była maksymalną bieżącą wartością i indeksem zaraz po poprzednim wystąpieniu bieżącego znaku. 

start = max(start, chars[char] + 1)
Nadal potrzebuję podpowiedzi.

Teraz musi Pan ustawić (zaktualizować) najnowszy indeks bieżącego znaku w słowniku. 

chars[char] = ind
Jak policzyć aktualną długość unikalnego podciągu?

Tak, ma Pan zapisaną wartość res i ma Pan aktualny indeks i wartość początkową (początek podciągu z unikalnymi znakami). Proszę znaleźć różnicę między indeksem a początkiem i zaktualizować res o maksymalną wartość. 

res = max(res, ind - start + 1)
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi, zwrócić końcowy wynik i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Oto całe rozwiązanie:

def longest_substr(s: str) -> int:

    start = res = 0
    chars = {}
    for ind, char in enumerate(s):
        if char in chars:
            start = max(start, chars[char] + 1)
        chars[char] = ind
        res = max(res, ind - start + 1)

    return res
