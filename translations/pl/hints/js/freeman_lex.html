Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę przechowywać aktualny początek podciągu i najdłuższą długość w zmiennych. Proszę utworzyć mapę do przechowywania ostatniego indeksu każdego znaku. Iteracja po podanym ciągu, aktualizacja pozycji początkowej w razie potrzeby (następny indeks po poprzednim wystąpieniu bieżącego znaku) i obliczenie długości unikalnych znaków.

Ok, od czego powinienem zacząć?

Proszę zadeklarować zmienne dla początku podciągu, najdłuższej długości i mapę znaków. 

let start = 0, maxLength = 0;
const charMap = new Map();

Proszę wykonać iterację po indeksach podanego ciągu za pomocą pętli for. 

for (let end = 0; end < s.length; end++) {
    ...
}
Co robić dalej?

Przede wszystkim należy sprawdzić, czy bieżąca postać jest obecna na mapie (czy została już wcześniej napotkana na podanym ciągu znaków). 

if (charMap.has(s[end])) {
    ...
}

Jeśli tak, należy zaktualizować pozycję start tak, aby była maksymalną bieżącą wartością i indeksem zaraz po poprzednim wystąpieniu bieżącego znaku. 

start = Math.max(start, charMap.get(s[end]) + 1);
Nadal potrzebuję podpowiedzi.

Teraz musi Pan ustawić (zaktualizować) najnowszy indeks bieżącej postaci na mapie. 

charMap.set(s[end], end);
Jak policzyć aktualną długość unikalnego podciągu?

Tak, ma Pan zapisaną wartość maxLength i ma Pan aktualny indeks i wartość początkową (początek podciągu z unikalnymi znakami). Proszę znaleźć różnicę między indeksem a początkiem i zaktualizować maxLength o maksymalną wartość. 

maxLength = Math.max(maxLength, end - start + 1);
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi, zwrócić końcowy wynik i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Oto całe rozwiązanie:

function longestSubstr(s: string): number {

    let start = 0, maxLength = 0;
    const charMap = new Map();

    for (let end = 0; end < s.length; end++) {
        if (charMap.has(s[end])) {
            start = Math.max(start, charMap.get(s[end]) + 1);
        }
        charMap.set(s[end], end);
        maxLength = Math.max(maxLength, end - start + 1);
    }
    return maxLength;
}

Przechowywanie unikalnych znaków w zestawie, przesuwanie indeksu początkowego, jeśli znak się powtarza, przesuwanie indeksu w lewo w przeciwnym razie. 

function longestSubstr(s: string): number {

    const charSet = new Set();
    let left = 0, right = 0, maxLength = 0;

    while (right < s.length) {
        if (!charSet.has(s[right])) {
            charSet.add(s[right]);
            maxLength = Math.max(maxLength, right - left + 1);
            right++;
        } else {
            charSet.delete(s[left]);
            left++;
        }
    }

    return maxLength;
}
